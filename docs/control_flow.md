# Control flow

## If-else

An `if` statement allows you to branch your code depending on conditions.

```
if (a %2 == 0) {
    // a is even
}
else {
    //a is odd
}
```

## While

A `while` statement allows you to run a block of code until a certain condition is met

```
while (a < 10) {
    a = a +1;
}
```

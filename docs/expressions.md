# Expressions

Expressions can be used in variable assignmentand in if and while conditions.

```
a = 34 + foo() + variable;      // 34 + foo() + variable is an expression!
```

Expressions can contain:

* Literals (32, 3.14)
* Variables (a, b, c)
* Function calls (foo())
* Operators
    * \+
    * \-
    * \*
    * \/
    * \=\= (equals to)
    * \!\= (not equals to)
    * \> (greater than)
    * \>\= (greater or equals)
    * \< (less than)
    * \<\= (less or equals)
